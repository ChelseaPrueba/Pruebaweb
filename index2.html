Software

Se denomina así al soporte lógico de un sistema informático, que comprende el conjunto de los componentes lógicos necesarios que hace posible la realización de tareas específicas. La interacción entre el software y el hardware hace operativo un ordenador (o cualquier dispositivo), es decir, el software envía instrucciones que el hardware ejecuta, haciendo posible su funcionamiento.

Los componentes lógicos incluyen las aplicaciones informáticas, tales como el procesador de texto, que permite al usuario realizar todas las tareas que compete a la edición de textos el llamado software de sistema, tal como el sistema operativo, que básicamente permite al resto de los programas funcionar adecuadamente, facilitando también la interacción entre los componentes físicos y el resto de las aplicaciones, y proporcionando una interfaz con el usuario.
